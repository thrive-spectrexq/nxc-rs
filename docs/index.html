Core capabilities

Built Different

Native Rust from the ground up — not a wrapper, not a port. Re-engineered for speed, stealth, and scale.

⚡
Async Runtime

Powered by Tokio's async runtime for massive concurrency. Scan full /16 subnets with configurable thread pools and minimal overhead.

🦀
Pure Rust

Zero Python. Zero Impacket. Native implementations of NTLM SSP, Kerberos, and SMB protocol stack baked directly into the binary.

🕶
Stealth & Safety

Built-in lockout detection, randomized jitter, and secure TLS communication to avoid triggering alarms during authorized assessments.

🤖
AI Orchestrator

Natural language mission control via LLMs. Chain tools, run recon, and automate entire engagement workflows with a single sentence.

🔍
AD CS & BloodHound

Integrated AD Certificate Services enumeration, BloodHound data export, and full Kerberoasting/AS-REP pipeline built into LDAP.

🗄
Workspace DB

Credential and target storage across sessions. Every hash, password, and finding is captured in structured workspace databases.

Get started

Installation

One-line install for most systems. Or build from source with Cargo.

Linux / macOS bash
# one-line install
curl -sSf https://raw.githubusercontent.com/
  thrive-spectrexq/nxc-rs/master/install.sh | bash
Windows powershell
# powershell one-liner
iex (New-Object System.Net.WebClient)
  .DownloadString('https://raw.githubusercontent.com/
  thrive-spectrexq/nxc-rs/master/install.ps1')
Cargo · requires rustc 1.94.0+ rust
# clone the repository
git clone https://github.com/thrive-spectrexq/nxc-rs.git && cd nxc-rs

# debug build (dev)
cargo build --workspace

# release binary
cargo build --release --package nxc

# binary location: target/release/nxc (Linux/macOS)
# target\release\nxc.exe (Windows)
WinRM
nxc winrm <target> -u Admin -p Pass123 -x "Get-Process"
Run PowerShell commands securely via PSRP over WinRM
SMB
nxc smb 192.168.1.0/24 -u Admin -H 31d6cfe0d16ae931b73c59d7e0c089c0 -M lsassy
Dump LSA secrets from an entire /24 subnet using pass-the-hash
LDAP
nxc ldap <dc> -u user -p pass --bloodhound --collection All
Export BloodHound data for full Active Directory attack path analysis
NFS
nxc nfs <target> --enum-shares
List exported NFS shares via the MOUNT service
LIST
nxc smb <target> -L
List all available offensive modules for a given protocol
